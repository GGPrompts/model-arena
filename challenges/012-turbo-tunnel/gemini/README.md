# Neon Turbo Rush

A Battletoads Turbo Tunnel inspired obstacle racer.

## How to Play
Open `index.html` in a modern web browser.

## Controls
- **Arrow Keys / WASD**: Change Lanes
- **Space / Up / W**: Jump (Avoid Gaps and Ramps)
- **Down / S**: Duck (Avoid Low Beams)

## Objective
Survive as long as possible. The speed increases every 10 seconds.
Watch for warning patterns and react quickly!
